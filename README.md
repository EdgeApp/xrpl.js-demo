# xrpl.js demo

## Installation

1. Clone the project
2. Run `npm install`

## Usage

Only edit `src/index.ts`. You can pass different arguments to the available functions.
Then run `npm run start`. The content of `index.ts` will be executed.

### Available functions

- sendPayment
- nftMint
- nftCreateOffer
- nftAcceptOffer
- nftCancelOffer
